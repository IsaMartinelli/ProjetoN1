# Proposta de Trabalho - Desenvolvimento Web

Este trabalho atribuirá uma nota ao aluno com o propósito de analisar seus conhecimentos
adquiridos ao longo do semestre sobre a disciplina "Desenvolvimento Web".
Não será permitido o uso de templates. O objetivo exige que somente os assuntos
abordados possam ser utilizados para a elaboração do trabalho.
Faça uma página que apresente uma galeria de filmes. Pode ser incluido cartaz do filme,
informações sobre os atores, diretores, tempo de duração etc. Não importa se é um lançamento ou
um filme antigo.
Divida a página como achar melhor. O que importa é a criatividade.
Uma sugestão é editar as imagens para que fiquem com as mesmas dimensões. Assim, a
configuração do CSS afeta as imagens da mesma forma.
